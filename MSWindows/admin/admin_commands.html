1.1 Changing forgotten login information - the easiest and fastest way

  1. boot a linux system to change the windows system files
  2. rename system32.exe in Utilman.exe_disa
  3. cp cmd.exe Utilman.exe
  4. reboot MSWindows
  5. press little keyboard button at the left lower corner of the login screen
  6. enter 'net user user_name new_password' (user_name: the user name you want to change)

Works on Windows7
